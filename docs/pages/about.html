Welcome to the alpha site for the Draft U.S. Web Design Standards.

Disclaimer: This site is actively under development. If you would like to make use of the Draft Web Design Standards and are having difficulty, please contact Mollie Ruskin at mruskin@omb.eop.gov.

The Draft Web Design Standards are intended to provide reference for designers, developers, as well as anyone else interested in the design and frontend principles that make up the building blocks of government websites.

Your feedback is important to us. We encourage you to contribute your input at {{ site.repos[0].url }}/issues.

The Draft U.S. Web Design Standards project is a partnership of the U.S. Digital Service and 18F with the support of partners across the government. If you'd like to learn more, please send a note to govtwidepatternlibrary@gsa.gov.
